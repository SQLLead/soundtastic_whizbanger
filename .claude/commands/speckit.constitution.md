Build a web-based sound board app that plays humorous sound effects when buttons are clicked.

Features:

- Grid layout of buttons (3-4 columns, responsive)
- Each button has a small icon and a label (e.g., "Fart", "Burp", "Knuckle Pop")
- Clicking a button plays the associated sound file
- Sounds include: flatulence, burping, knuckle cracking, neck crunch, back creak, head bonk, "sproing" sound, cough, sneeze
- Playful, cartoonish visual design
- Works on mobile and desktop
- Easy to add new sounds: just add MP3 file and update a simple array in the code

Technical:

- Single HTML file with embedded CSS and JavaScript
- Sounds stored in /sounds/ subdirectory
- No external dependencies or libraries
